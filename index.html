FREQUENTLY ASKED QUESTIONS

  • What customization does Predator offer?
    Predator offers support for top/bottom toolbar, light/dark theme or automatic theme adjust according to device theme.
  • Does Predator access the user's information?
    No! Predator is a lightweight privacy-focused browser, Predator doesn't access your browsing data of course as well as it doesn't receive the user's usage.
  • What makes Predator fast?
    Predator is based on GeckoView which is currently the fastest browsing back end, also it blocks ads and doesn't autoplay videos to save data and load time.
  • When will Predator be available to download?
    Predator is still in-development and in alpha stage, it's planned to be available within the next two months.
  • What's the required specs to run Predator?
    Android OS: 5.0+
    RAM: 512MB+
    Size: 70MB
  • Does it support extensions/add-ons?
    Yes, Predator supports extensions, simply click on menu button in toolbar, choose Extensions to manage all the extensions.
  • Is Predator a good choice?
    Predator has everything you need to switch it, from private browsing and bookmarks to top/bottom toolbar and extensions. So the answer is yes, Predator is ready to be used as your daily browser.
  • What new features are in-development?
    The next feature is a new homepage with gaming news, homepage will have the background of your device. Also the biggest feature is the ability to design your own theme , many other features are ram usage display and limiter, in-browser sound effects, and out of the box custom themes.
  • Will Predator release on other platforms?
    There's currently a Windows version which is still in early stages of development which will be released later this year. Also a linux version is planned to be developed.
